「TextAlive App API」とは

楽曲中の歌声に合わせて歌詞をアニメーションさせるリリックビデオの制作支援サービス「TextAlive（テキストアライブ）」の機能を使って、歌詞が音楽に合わせてタイミングよく動くWebアプリケーション（リリックアプリ）を開発できるJavaScript用のライブラリです。一般的なWebアプリケーション用の開発環境で、必要に応じて three.js, PixiJS, p5.js など好みのライブラリと組み合わせながら、自由な映像表現をプログラミングできます。

※ TextAlive および TextAlive App API は、産業技術総合研究所が研究開発をして一般公開しています。

審査基準

主催側による入選作品の審査は、以下の3つの観点で行います。

クリエイティビティ
Webアプリケーションの演出が楽曲と同期して魅力的に見えるか
イノベーション
Webアプリケーションを支えるアイデアがユニークで、未来の創作文化を予感させるものであるか
完成度
Webアプリケーションが一般的なWebブラウザ（パソコン、タブレット端末、スマートフォンなど応募者の意図した環境）で正常に動作するか、実装が技術的に優れているか
審査の流れ
  1. 応募締め切り後、主催側により審査で入選作品を選定します
  2. 入選作品を「マジカルミライ 2024」公式Webサイトで公開し、情報アプリ『初音ミク公式ナビ（通称：ミクナビ）』にてユーザー投票を行います
  3. 投票結果を参考に入賞作品を決定します
  4. 「マジカルミライ 2024」TOKYO会場の企画展ステージで入賞作品を発表します

応募について

応募形式

Webアプリケーションのソースコード（HTML、CSS、JavaScriptなど）

  • HTTPサーバ上に設置するだけで動作する静的アプリケーションのみとします（PHP、Ruby、Node.jsなどを利用した動的なレスポンスを返すサーバアプリケーションは不可）
  • ソースコードのビルド用スクリプト（Webpack, Parcel, Viteなど一般的なビルドツールを利用したもの）は含めて構いません
  • Webアプリケーションをソースコードからビルドしてパソコン用のWebブラウザ、タブレット端末、スマートフォンなど応募者の意図した環境に近いものを用いて審査予定ですが、参考情報としてデモ動画のURLや応募者自身でホストしたWebサイトのURLを提出いただけます

FAQ

応募作品中で、初音ミク等の画像を利用してよいのでしょうか？

いわゆる二次創作作品については、「ピアプロキャラクターズ」のみ登場させることができます。これに限らず画像をお使いいただくことは可能ですが、必ず

  • ご自身で制作したもの
  • 正規に利用の許諾を受けたもの

のいずれかをお使いいただき、主催者に対して権利面で問題がないことを保証していただきます。

また、正規に利用の許諾を受けた作品をお使いいただく際は、その作品の作者のお名前をアプリ中に明示的かつ合理的な方法で記載するようにしてください。

応募作品中で、対象楽曲以外の効果音等も鳴らしてよいのでしょうか？

効果音等を鳴らすことは可能ですが、以下の点を遵守していただきます。

  • 必ずご自身で制作したものか、正規に利用の許諾を得た効果音をお使いいただき、主催者に対して権利面で問題がないことを保証していただきます。
  • 効果音が対象楽曲の一部を構成するような印象を与える使用はお控えください。
応募締め切り後にGitHubのプライベートリポジトリへコミットしてもよいでしょうか？

審査時には応募締め切り時点での最終コミットを採用します。したがって、コミットしても審査には影響しません。
